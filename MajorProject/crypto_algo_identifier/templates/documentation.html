Supported Algorithms

AES (Advanced Encryption Standard)

Type: Symmetric Encryption

Key Sizes: 128 / 192 / 256 bits

Use Cases:

  • SSL/TLS encryption
  • Disk encryption (BitLocker, FileVault)
  • Wireless security (WPA2/WPA3)

Key Features:

  • Substitution-Permutation Network (SPN)
  • High security and performance
  • Uses Galois Field arithmetic

DES (Data Encryption Standard)

Type: Symmetric Encryption

Key Size: 56 bits

Use Cases:

  • Legacy systems
  • Historical cryptography studies

Key Features:

  • Feistel network structure
  • Fast in hardware
  • Vulnerable to brute-force attacks

3DES (Triple DES)

Type: Symmetric Encryption

Key Size: 112 / 168 bits

Use Cases:

  • Banking systems
  • Legacy financial applications

Key Features:

  • Applies DES three times
  • Improved security over DES
  • Slow compared to AES

Blowfish

Type: Symmetric Encryption

Key Size: 32 – 448 bits

Use Cases:

  • Password hashing systems
  • File encryption tools

Key Features:

  • Fast and efficient
  • Feistel network design
  • Flexible key length

MD5

Type: Hash Function

Output Size: 128-bit hash

Use Cases:

  • Checksum verification
  • Legacy hashing systems

Key Features:

  • Fast computation
  • Produces fixed-length hash
  • Broken due to collisions

SHA-256

Type: Cryptographic Hash Function

Output Size: 256-bit hash

Use Cases:

  • Blockchain (Bitcoin)
  • Digital signatures
  • SSL/TLS security

Key Features:

  • Strong collision resistance
  • Part of SHA-2 family
  • Widely trusted standard

RSA

Type: Asymmetric Encryption

Key Sizes: 2048 / 3072 / 4096 bits

Use Cases:

  • Secure key exchange
  • Digital signatures
  • SSL/TLS certificates

Key Features:

  • Public and private key pair
  • Based on prime factorization
  • Highly secure but slower

ECC (Elliptic Curve Cryptography)

Type: Asymmetric Encryption

Key Sizes: 224 / 256 / 384 / 521 bits

Use Cases:

  • Mobile security
  • Cryptocurrencies
  • Secure messaging apps

Key Features:

  • Smaller key sizes
  • Faster computations
  • High security efficiency
